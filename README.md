# Calculatrice Web Simple

# Aperçu du Projet
Ce projet est une application de calculatrice simple et fonctionnelle, développée dans le cadre d'un portfolio. Elle permet d'effectuer les quatre opérations arithmétiques de base (addition, soustraction, multiplication et division), ainsi que d'autres fonctions essentielles.
L'objectif principal est de démontrer la maîtrise de la logique JavaScript et la manipulation du DOM (Document Object Model).

# Fonctionnalités
La calculatrice prend en charge les actions suivantes :

* **Opérations de base :** Addition (+), Soustraction (-), Multiplication (x), Division (÷).
* **Affichage :** Mise à jour en temps réel de l'opération en cours.
* **Décimales :** Prise en charge des nombres décimaux.
* **Nettoyage total :** Bouton `AC` (All Clear) pour réinitialiser l'affichage.
* **Suppression :** Bouton `DEL` pour supprimer le dernier chiffre entré.
* **Négatif/Positif :** Bascule entre nombres positifs et négatifs.

# Technologies Utilisées
Ce projet est entièrement basé sur le front-end et utilise les technologies suivantes :

* **HTML5 :** Pour la structure et la mise en page de la calculatrice.
* **CSS3 :** Pour le style, le design des boutons et le responsive design.
* **JavaScript (Vanilla JS) :** Pour la logique de calcul et la manipulation des événements.

# Installation et Utilisation
Puisqu'il s'agit d'un projet purement front-end, l'installation est très simple.

### Lancement Local

1.  Clonez le dépôt :
    ```bash
    git clone https://github.com/mcstakyra/calculatrice.git)
    cd calculatrice
    ```
2.  Ouvrez le fichier `index.html` directement dans votre navigateur (Firefox, Chrome, Edge, etc.).
